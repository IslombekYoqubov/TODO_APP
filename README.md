# TODO App

A simple and clean **Android TODO application** designed to help users manage daily tasks efficiently. The project follows modern Android development practices and focuses on readability, scalability, and user experience.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Clean and minimal UI

## Tech Stack

- **Language:** Kotlin
- **Platform:** Android
- **Architecture:** MVVM
- **UI:** XML
- **Data Storage:** Room Database

---

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/IslombekYoqubov/TODO_APP.git
   ```
2. Open the project in **Android Studio**
3. Sync Gradle files
4. Run the app on an emulator or physical device
